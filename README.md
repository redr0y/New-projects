This password generator I made to be used at my first internship. When creating accounts or reseting passwords it can be tedious thinking of a unique password, which is why I made this

Yes there are millions of other password generators out on the internet, but I had trouble finding one that did exactly what I wanted it to do, and I thought it would be fun to make

I wanted the password generator to try to always use whole words. Lots of the ones I found online would scramble up the letters numbers and special character, leading to the password -
- being very difficult to remember. We need our passwords to be more secure as hackers pose more of a threat to us. However, I want whoever uses this generator to have a easy time -
- remembering their new password. So, this generator makes the password easy to remember but also secure.
